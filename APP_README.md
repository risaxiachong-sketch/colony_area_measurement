# シャーレ菌叢 解析アプリ

`streamlit_app.py` は、シャーレ写真から菌叢の投影面積を測るための Streamlit アプリです。

## できること

- 複数の画像をまとめてアップロードして解析
- 画像フォルダを ZIP にしてアップロードして解析
- 自分のPCで動かす場合は、PC内フォルダのパスを指定して一括解析
- シャーレ直径、色の拾いやすさ、中心部の拾いやすさを画面で調整
- 結果CSVとオーバーレイ画像をダウンロード

## ローカルPCで動かす方法

最初に必要なライブラリを入れます。

```powershell
pip install -r requirements.txt
```

アプリを起動します。

```powershell
streamlit run streamlit_app.py --browser.gatherUsageStats false
```

ブラウザで `http://localhost:8501` が開きます。

## 入力方法

### 画像またはZIPをアップロード

画像を複数選ぶか、画像フォルダを ZIP にしてアップロードします。

この方法は、Web公開してみんなで使う場合に向いています。Webアプリは利用者のPC内フォルダを直接読むことができないためです。

### PC内フォルダのパスを指定

自分のPCでアプリを動かしているときだけ使えます。

エクスプローラーで画像フォルダを開き、アドレスバーのパスをコピーして貼り付けます。

```text
C:\Users\risa4\OneDrive\...\iCloud写真
```

## みんなが使えるようにする方法

おすすめは Streamlit Community Cloud で公開する方法です。

1. GitHubにこのリポジトリを置く
2. `streamlit_app.py`、`measure_colony_area.py`、`requirements.txt` を入れておく
3. Streamlit Community Cloud で GitHub リポジトリを選ぶ
4. Main file に `streamlit_app.py` を指定する
5. 公開URLを研究室メンバーに共有する

Web公開した場合は、利用者は「画像またはZIPをアップロード」を使います。

## 菌の種類が変わる場合

菌の色や形が変わると、同じ設定ではうまく拾えないことがあります。
そのため、アプリには解析目的ごとのプリセットがあります。

- 標準（黄色・黒色中心）
  - これまでのスコポレチン培地のように、黄色〜ベージュの菌糸と黒い中心部を拾います。
- 白色菌糸を拾う
  - 白い枝状の菌糸を拾いやすくします。
- 茶色色素も含める
  - 菌糸だけでなく、茶色〜黄色に変色した培地部分も含めます。
- 外縁面積を測る
  - 枝のすき間をある程度埋めて、菌叢が広がった外側の範囲を測ります。

最初はプリセットを選んで解析し、オーバーレイ画像を見て、拾いすぎ・拾わなさすぎを確認してください。
必要な場合だけ、サイドバーの「詳細設定」を調整します。

## 別の配布方法

- Google Colab: 学習用・授業用に向いています。ノートブックをコピーして使えます。
- ローカル実行: 研究室PCだけで使うなら安定します。フォルダパス指定が使えます。
- Windowsアプリ化: ダブルクリックで開ける形にもできますが、最初の作成と更新管理が少し難しくなります。

## 主に見る値

- `mycelium_mm2`: 白からベージュの菌糸部分の面積
- `dark_core_mm2`: 中央の黒っぽい部分の面積
- `colony_mm2`: 菌糸部分と黒い中心部を含む合計面積

## 注意

測定しているのは、写真を上から見た投影面積です。立体的な表面積そのものではありません。
